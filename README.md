# cpp

## Description
c++ programming language

## contemporary c++
Contemporary C++ isn't just novel features

- Some key features and techniques are old, e.g.,
  - Classes with constructors and destructors
  - Exceptions
  - Templates
  - std::vector

- Some are more recent, e.g.,
  - Modules
  - Concepts (for specifying generic interfaces)
  - Lambda expressions (for generating function objects)
  - Ranges
  - Constexpr and consteval (for compile-time computation)
  - Concurrency support and parallel algorithms
  - Coroutines (missing for decades though they were an essential part of early C++)
  - std::shared_ptr
